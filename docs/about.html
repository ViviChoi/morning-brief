Morning Brief

A daily market intelligence snapshot. Updated every morning at 08:00 Europe/Rome by a GitHub Actions cron — no servers, no logins.

What's on the home page

The landing page is today's snapshot of US-equity + crypto markets, paired with a 0–100 BTC bottom-score and a plain-language commentary written by Claude. Built for people who don't normally read finance — charts dominate, every number is annotated, color is consistent (green = up, red = down).

How it's built

  • Data: yfinance, CoinGecko, alternative.me (Crypto F&G), farside.co.uk (BTC ETF flows), Binance public futures. All free.
  • AI commentary: Anthropic Claude (Sonnet 4.6). One run = ~$0.05.
  • Schedule: GitHub Actions cron daily; manual trigger from the Actions tab.
  • Hosting: GitHub Pages, deployed via official Actions pipeline. No servers to maintain.

Full source: github.com/ViviChoi/morning-brief — MIT-licensed.

Two surfaces in the project

Daily Brief (what you see on the home page): public-shareable, static HTML snapshot.

Portfolio Explorer: a separate local FastAPI tool the owner runs at home — drag-drop your broker CSV, see allocation donut, P&L bars, and a plain-language portfolio explanation. Not deployed publicly because uploading broker statements to a public server is a privacy bad-idea. See the README on GitHub if you want to run it yourself.

Disclaimer: This is a personal engineering project. The AI commentary is algorithmically generated from public market data and is not investment advice. Don't trade on the basis of what a 0–100 score says — it's a tool for tracking sentiment, not for buying decisions.
